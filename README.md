# Interactive Body Atlas

Интерактивный 3D-атлас тела на Three.js: просмотр GLTF-модели, управление группами анатомии по CSV и маркерами на сцене.

## Что это

- **Просмотр модели** — загрузка одной или двух частей модели (`.glb`), по умолчанию всё скрыто; показ/скрытие по группам из CSV.
- **Группы из CSV** — нумерованные группы (только чтение) и редактируемые подгруппы (`.g`); загрузка/сохранение CSV.
- **Маркеры** — добавление маркеров по клику на меш или по координатам; группы маркеров, экспорт/импорт JSON.
- **Скрипты в HTML** — команды для сцены и данных задаются в разметке (например, в теге `<script type="text/atlas-script">` или в обычном `<script>`), а не в консоли браузера.

## Модули

| Файл | Назначение |
|------|------------|
| `main.js` | Точка входа, реэкспорт `CSVGroupsManager`, `AtlasScene`, `MarkerManager`, `MarkerUI`. |
| `AtlasScene.js` | 3D-сцена (Three.js), камера, загрузка GLTF и CSV, индекс мешей по имени, группы, материалы. |
| `csv-groups.js` | Парсинг/сериализация CSV групп, `CSVGroupsManager` (подгруппы, видимость, сохранение). |
| `MarkerManager.js` | Маркеры на сцене (сфера/куб), привязка к мешам или координатам, группы, JSON. |
| `MarkerUI.js` | Панели и формы: группы, маркеры, клики по канвасу, попап, таблица маркеров. |

Подробное оглавление API — в JSDoc в начале каждого файла.

## Запуск

Откройте в браузере файл **`index.html`** (двойной клик или «Открыть файл»).

## Использование

- **Группы** — чекбоксы в панели «Группы» показывают/скрывают анатомические группы из CSV.
- **Маркеры** — включить режим «Добавить маркер», кликнуть по видимой части модели; редактирование в панели «Данные маркера».
- **Команды в HTML** — вызовы API сцены (`show_group`, `hide_group`, `load_csv`, `get_groups` и т.д.) нужно прописывать в скриптах внутри `index.html`, а не вводить в консоли браузера.
